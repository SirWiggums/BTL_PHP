# BTL_PHP - Website Bán Máy Tính

🌟 **Giới thiệu**  
BTL_PHP là một dự án website bán máy tính chuyên nghiệp, mang lại giải pháp thương mại điện tử tối ưu cho người dùng.

🌐 **Tính năng**  
- 🖥️ Tìm kiếm và hiển thị sản phẩm.  
- 🛒 Giỏ hàng tiện lợi.  
- 💳 Thanh toán an toàn.  

🔧 **Công nghệ sử dụng**  
- PHP  
- MySQL  
- Bootstrap  

📦 **Cách cài đặt**  
1. Tải mã nguồn từ GitHub.  
2. Cài đặt XAMPP và tạo cơ sở dữ liệu.  
3. Chạy website qua localhost.
